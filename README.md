## Quick Start

Morpho Markets indexer is built using [Ponder](https://ponder.sh/).

To start it:

1. Install dependencies:

```bash
yarn install
```

2. Run the codegen:

```bash
yarn codegen
```

3. Run the indexer:

```bash
yarn start
```
